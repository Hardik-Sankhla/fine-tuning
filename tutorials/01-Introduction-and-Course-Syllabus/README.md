# Session 01: Introduction and Course Syllabus

## Fine-Tuning LLMs: Course Kickoff

This opening session introduces the end-to-end roadmap for modern LLM fine-tuning, from supervised adaptation to preference alignment methods such as RLHF and DPO.

You will learn what problems fine-tuning solves, how it differs from pretraining, and how the techniques in this course connect into a practical training pipeline.

## Learning Outcomes

- Understand the scope and structure of the full course.
- Identify where LoRA, QLoRA, RLHF, and DPO fit in production workflows.
- Set expectations for compute, data quality, and evaluation requirements.
